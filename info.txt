---------

1)copiar package.json

2)npm i -g nodemon //--> instala a nivel global (en la pc)

3)yarn

4)yarn add compression

5)yarn add express-sslify //-->para encriptar https (paso 1 de PWA)

//------mercadopago--------//

------
Notas: 
------
        !!Activar lineas de server.js al pasar a Heroku!!!

*)nodemon:    
    Biblioteca que permite crear el server.
    Actualiza el servidor ante cualquier cambio.

*)"engines"-> key: donde la version de node y npm
        1)node -v -> copiar numero
        2)npm -v -> copiar numero

*)express: framework para hacer el backend
            En el servidor no se usa es6, ya que no tiene un interprete como Babel,
            por esto se importa la biblioteca de otra manera

*).env -> archivo donde va la clave secreta. 
            add a .gitignore como .env -> para evitar mandar informacion de seguridad.

*)"proxy": "http://localhost:5000" -> se coloca en json de client.
            Permite unir el front con el back. 

*)Cors: cross origin request.
            Cruza los puertos. Ej el 3000 del front con el 5000 del back, 
            Une servidores ??

*)Token: clave que permite a la api(stripe) identifica de donde se hace la peticion.
